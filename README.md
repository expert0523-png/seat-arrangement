# 🏫 國小座位編排工具

國小教師專用的座位編排工具，支援一般座位與分組座位排列。

## ✨ 功能特色

- 📋 **學生名單管理** — 座號、姓名、性別，支援逐筆新增或批次匯入
- 🚫 **約束設定** — 設定不能相鄰的學生、必須坐第一排的學生
- 🎲 **自動排位** — 系統自動依約束條件排定座位
- ✋ **手動微調** — 拖放交換學生座位
- 📐 **一般座位** — 可調整行列數（預設 4×4）
- 👥 **分組座位** — 可調整組數（預設 4 組）
- 💾 **資料儲存** — 自動儲存至瀏覽器，支援 JSON 匯出入
- 🖼️ **匯出功能** — 匯出為 PNG 圖片或列印

## 🚀 部署到 GitHub Pages

這是一個純靜態網頁應用，無需伺服器或資料庫。您可以免費將其部署在 GitHub Pages。

### 步驟 1：建立 GitHub 專案
1. 登入 GitHub 並點擊右上角的 `+` 號，選擇 **New repository**
2. 自訂一個專案名稱（例如：`seat-arrangement`）
3. 設定為 **Public**，點擊 **Create repository**

### 步驟 2：上傳檔案
1. 點選畫面上的 `uploading an existing file`
2. 將 `index.html` 與本說明檔 `README.md` 拖曳上去
3. 點擊 **Commit changes**

### 步驟 3：開啟 Pages 功能
1. 點擊上方的 **Settings** 標籤
2. 點擊左側導覽列的 **Pages**
3. 在 **Build and deployment** 下方的 **Branch** 選單選擇 `main`，然後點擊 **Save**
4. 稍等數分鐘，頁面頂端會顯示您的專屬網址！

直接在瀏覽器中雙擊開啟 `index.html` 即可使用，無需安裝任何軟體。

## 🎯 使用教學

1. **新增學生**：在左側「名單」面板輸入座號、姓名、性別，點擊「新增」
2. **批次匯入**：點擊「批次匯入名單」，從 Excel 複製貼上（格式：座號,姓名,性別）
3. **設定約束**：切換到「約束」面板，設定不能相鄰或必須坐第一排的學生
4. **配置座位**：在工具列調整行列數（一般模式）或組數（分組模式）
5. **自動排位**：點擊「🎲 自動排位」
6. **手動微調**：拖放學生卡片交換位置
7. **匯出**：點擊「📤 匯出」下載 JSON 設定檔或圖片

| 快捷鍵 | 功能 |
|--------|------|

| `Esc` | 關閉彈出視窗 |
